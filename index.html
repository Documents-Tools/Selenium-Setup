Selenium Setup

Start your own Selenium test server with Chrome, Firefox and Internet Explorer drivers.
If you are a PHP developer we have a helper included for you to write tests with Facebook's WebDriver.

Start Selenium Server on Windows, Linux or Mac.

Selenium Setup will attempt to:

  • detect your environment.
  • check the pre-requisites.
  • everything you need in your server instance.

A simple command line can get you up and running in no time:

./selenium-setup start
See the CLI Demo

How? See for yourself.

We tried to narrow the tech stack needed to run the whole setup.

./selenium-setup register myFirstServer 4444
./selenium-setup register mySecondServer 4445
./selenium-setup register myThirdServer 4446
...

Why? Testing your code is important!

There are a couple of reasons this library was written, we needed the following:

  • a quick way to start Selenium Server without the hassle involving the web drivers
  • a way to configure every detail about the Selenium Server we want through default.json
  • give the opportunity to PHP devs to focus on writing acceptance tests with Facebook's WebDriver
  • raise multiple Selenium Server instances on both Windows and Linux environments
